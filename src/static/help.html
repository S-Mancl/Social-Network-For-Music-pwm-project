Help page!
Due to the high requests, we decided to include an explanation page. Here you will be able to search for various features. Every page will have a link to this page in the upper right corner of the screen.
# Search
Using this feature, you can browse through every track, album, episode, artist or even show that is available. You need to insert a string to search for and select some categories, and then press the button.
You will see the first five results for every category, and by pressing "Show more" you'll be able to see others, and browse through multiple pages.
# Describe
From this page, you can view more details about your search. If you're logged, you can add it to your favorites and, if it is a track, even to a playlist! WOW!
# Playlists
With this page, you can browse public playlist, searching them by name or by tag, and you can create new playlists.
# Explain a playlist
This page allows you to edit a playlist data (if you're the owner) or to follow/unfollow it. It even allows you to insert it into a group.
# Groups
With this page, you can browse groups, filtering them by name. You can also create a new group.
# Explain a group
This page allows you to edit a group data (if you're the owner) or to join/leave it.
# Login
With this page, you can login into your account. If you don't have an account, consider registering yourself for free.
# Register
With this page, you can register a new account. If you already registered, you can instead login.
# My Profile
From this page, you can see and edit your personal data, your favorites, playlists, groups and other details.
